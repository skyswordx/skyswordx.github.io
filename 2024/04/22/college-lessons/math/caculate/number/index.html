级数

发表于2024-04-22|更新于2024-04-29
|字数总计:1.9k|阅读时长:7分钟|阅读量:
笔记来源 讲课老师
中山大学工科高数课堂 @杨奇林

收敛的级数具有线性性，两个收敛的级数的线性组合、数乘还是收敛的

对一个收敛的级数在前面若干项做变动是不会影响级数的收敛性的

等比级数

如果比例为 r\displaystyle rr，那么等比级数的部分和为 Sn=a1−anr1−r\displaystyle S_{n}=\frac{a_{1}-a_{n}r}{1-r}Sn​=1−ra1​−an​r​ (首项\displaystyle \frac{首项}{}首项​)
如果 ∣r∣<1\displaystyle \mid r\mid< 1∣r∣<1 那么等比级数的和为 a11−r\displaystyle \frac{a_{1}}{1-r}1−ra1​​ (首项1−公比\displaystyle \frac{首项}{1-公比}1−公比首项​)
如果 r=1\displaystyle r=1r=1，则级数和为 ∞\displaystyle \infty∞，不收敛
如果 r=−1\displaystyle r=-1r=−1，级数和也不收敛

正项级数收敛

正项级数收敛的充要条件是它的部分和 Sn\displaystyle S_{n}Sn​ 有上界

正项级数收敛，它的子列也收敛
正项级数的子列收敛，它本身也收敛

1np\displaystyle \frac{1}{n^{p}}np1​ 级数

这个 ∑n=1∞1np\displaystyle \sum \limits_{n=1}^{\infty}\frac{1}{n^{p}}n=1∑∞​np1​ 级数，当 p>1\displaystyle p>1p>1 时收敛，当 p≤1\displaystyle p\leq 1p≤1 时发散
经常利用这个级数去拿你要证明的级数进行比较判别法辅助判定

比较判别法

核心就是如果要证明一个复杂级数的收敛性，就要找到另外一个好算的级数来辅助
判别的方法就是两个级数，大的如果收敛，小的也一定收敛；小的如果发散，大的也一定发散

除了直接凑比大小，还可以使用泰勒展开来得到等价的级数。因为这个级数的收敛其实就是无穷小的运算，分析通项是不是趋近于 0，如果不趋近于 0 那根本不可能收敛，如果趋于 0 就是无穷小量，可以使用泰勒展开。要注意可以通过泰勒展开那个原来级数的式子，找到和原来级数等价的多项式展开式。这样子就可以用简单的等价多项式辅助证明原来复杂的级数了

具体的验证方法就是，已知一个复杂的级数 un\displaystyle u_{n}un​ 我们用泰勒展开构造了一个好算的级数 vn\displaystyle v_{n}vn​ 接下来要去验证这二者是否可以等价，就是看 lim⁡n→∞unvn\displaystyle \lim_{ n \to \infty } \frac{u_{n}}{v_{n}}n→∞lim​vn​un​​ 是否等于常数，如果等于常数，就找到了等价的好算级数 vn\displaystyle v_{n}vn​ 这时候 un\displaystyle u_{n}un​ 和 vn\displaystyle v_{n}vn​ 的收敛性是一样的

在这里经常会遇见 1np\displaystyle \frac{1}{n^{p}}np1​ 还有一些类似的变形，比如 nun=un1n\displaystyle nu_{n} = \frac{u_{n}}{\frac{1}{n}}nun​=n1​un​​ 以及 npun=un1np\displaystyle n^{p}u_{n}= \frac{u_{n}}{\frac{1}{n^p}}npun​=np1​un​​
或者知道 npanp\displaystyle n^{p}a_{n}^{p}npanp​ 收敛时，可以作为条件用极限的拆分运算推出 an\displaystyle a_{n}an​ 收敛

比值判别法

lim⁡n→∞un+1un=后一项前一项=l\displaystyle \lim_{ n \to \infty } \frac{u_{n+1}}{u_{n}} = \frac{后一项}{前一项}=ln→∞lim​un​un+1​​=前一项后一项​=l 本质上是看后一项比前一项的比值是大于 1 还是小于 1
其实就是拿这个级数等效成等比级数，如果等效出来的比例小于 1，那等比级数就是收敛的嘛

lim⁡n→∞un+1un=后一项前一项=l{l<1是衰减的，收敛l>1不是衰减的，发散l=1换方法吧\displaystyle \lim_{ n \to \infty } \frac{u_{n+1}}{u_{n}} = \frac{后一项}{前一项}=l \begin{cases}l<1 \quad 是衰减的，收敛\\ l>1 \quad 不是衰减的，发散\\ l=1 \quad 换方法吧\end{cases}n→∞lim​un​un+1​​=前一项后一项​=l⎩⎨⎧​l<1是衰减的，收敛l>1不是衰减的，发散l=1换方法吧​

这个比值判别法适合阶乘、多项式和常数的 n\displaystyle nn 次方连乘的情况

在做比值的时候，要考虑两个重要的极限 sin⁡x=x\displaystyle \sin x=xsinx=x 和 (1+1n)n【1的无穷大】=e\displaystyle \left( 1+\frac{1}{n} \right)^n【1的无穷大】=e(1+n1​)n【1的无穷大】=e

遇到比值为 1，就换 lim⁡n→∞un+1un=1+pn\displaystyle \lim_{ n \to \infty } \frac{u_{n+1}}{u_{n}}=1+\frac{p}{n}n→∞lim​un​un+1​​=1+np​ 泰勒展开

lim⁡n→∞(un)1n=l{l<1是衰减的，收敛l>1不是衰减的，发散l=1换方法吧\displaystyle \lim_{ n \to \infty }(u_{n})^{\frac{1}{n}} =l \begin{cases}l<1 \quad 是衰减的，收敛\\ l>1 \quad 不是衰减的，发散\\ l=1 \quad 换方法吧\end{cases}n→∞lim​(un​)n1​=l⎩⎨⎧​l<1是衰减的，收敛l>1不是衰减的，发散l=1换方法吧​

柯西判别法适合处理 n\displaystyle nn 的 n\displaystyle nn 次方的

级数和对应的函数具有相同收敛性

一个 x\displaystyle xx 连续变化的函数 f(x)\displaystyle f(x)f(x) 和一个 n\displaystyle nn 离散变化的级数 an=f(n)\displaystyle a_{n}=f(n)an​=f(n) 是具有相同收敛性的

写成公式就是 ∑n=1∞an=∫1Af(x) dx\displaystyle \sum \limits_{n=1}^{\infty}a_{n}=\int_{1}^{A} f(x) \, dxn=1∑∞​an​=∫1A​f(x)dx 一旦级数 an\displaystyle a_{n}an​ 收敛，那么积分结果 ∫1Af(x) dx=P(A)\displaystyle \int_{1}^{A} f(x) \, dx =P(A)∫1A​f(x)dx=P(A) 在 A→∞\displaystyle A\to\inftyA→∞ 的时候极限存在

所以遇到难算的级数可以把 n\displaystyle nn 改成 x\displaystyle xx 用函数积分计算
具体来说就是计算这个函数积分得到 ∫1Af(x) dx=P(A)\displaystyle \int_{1}^{A} f(x) \, dx =P(A)∫1A​f(x)dx=P(A) ，再看 A→∞\displaystyle A\to \inftyA→∞ 之后，P(A)\displaystyle P(A)P(A) 存不存在，如果存在就是收敛，不存在就发散
